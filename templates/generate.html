In order to analyze your own comments, you need to create a CSV file with all of the comments you wish to analyze. The easiest way to do this is using BYU's student rating system.

Step 1:

Go to https://ratings.byu.edu/. Click on "View Student Ratings reports".

Step 2:

Under the section titled "Summative Student Ratings Reports", click on the button titled "View Report". This will bring up a pop-up. Select the first semester which you wish to analyze; the report will automatically contain all comments from then until your most recent class.

Step 3:

The result will be a page that contains a summative report at the top. Below the summative report is a section containing all of your comments. There is a button titled "Download comments in .csv format". Click the button, and save the resulting file. That's the file you need!
