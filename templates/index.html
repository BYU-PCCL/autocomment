Welcome!

This is an automatic student comment analyzer based on various machine learning and natural language processing techniques. Click on the links in the navigation pane to the left to see a variety of visualizations designed to convey insight about large numbers of student comments.

At the heart of this system is an analysis of the sentiment and magnitude of each comment. See the image at the right for an example. Each sentence in a comment is assigned a sentiment, which is a numeric score indicating if the sentence conveys a negative sentiment or a positive sentiment. Numeric values range from -1 to +1. An overall sentiment score for the entire coment is assigned by combining the sentiment scores for individual sentences.

Similarly, sentences and overall comments are assigned a magnitude, which represents the "strength" of the sentiment. Some comments can be unambigiously positive, such as "Dr. Smith is a great professor.", but which aren't passionate or particularly informative. It is therefore useful to look for comments that have extreme values for sentiment (ie, close to +1 or -1), as well as a large magnitude.

Most of the visualizations in this system are derived by sorting, filtering, categorizing, and visualizing some combination of sentiment and magnitude. Some visualizations invoke additional textual analysis or machine learning algorithms.

More information about the specific technical methodology is available on our about page.
